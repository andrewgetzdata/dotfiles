# /evening — Night Prep

Wind down and set up tomorrow for success.

## Procedure

### 1. Check tomorrow's calendar

Use Google Calendar MCP tools:
1. `list-events` for tomorrow — meetings, commitments, time blocks
2. Flag anything that needs prep (materials, agendas, context)

### 2. Review tomorrow's tasks

Use Todoist MCP tools:
1. `find-tasks-by-date` for tomorrow
2. Highlight priorities and any deadlines

### 3. Prep for meetings

For each meeting tomorrow:
- Search `meetings/` and `people/` for prior context with attendees
- Search `knowledge/` for relevant topic notes
- Briefly note what context exists — "You last met with X on Y about Z"

### 4. Prompt for reflection

Ask Andrew:
- How was today? (gratitude, energy, anything on your mind)
- Anything to capture before bed?

Save responses to today's journal entry under a `## Reflection` section. Update the habit fields in frontmatter if Andrew provides them (gratitudeHabit, exerciseHabit, journalHabit).

### 5. Present the evening brief

Print a concise look-ahead:
- **Tomorrow's schedule**: Key events and times
- **Top priorities**: What matters most tomorrow
- **Meeting prep**: Context for tomorrow's meetings (people, topics)
- **Knowledge queue**: How many articles are pending (gentle reminder, not a task)

Keep the tone relaxed — this is end-of-day, not a status report.

## Tools Used

- Google Calendar MCP: list-events
- Todoist MCP: find-tasks-by-date
- Read, Write, Edit, Glob, Grep (file operations)
